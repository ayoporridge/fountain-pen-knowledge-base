# 钢笔的意义何在？

# 直击核心：钢笔的意义何在？
（本文发布于2008年11月1日）
（本文修订版首次刊登于2006年2/3月号《Stylus》杂志）

---

谁还需要钢笔呢？说真的，如果你想引人注目，或是搭配你的连衣裙、衬衫或西装，又或者你只是不满足于BiC圆珠笔——为什么不用中性笔呢？多数优质钢笔品牌都生产与自家钢笔外观相似的中性笔，何必自找麻烦多花钱买钢笔？

事实上，使用钢笔有诸多充分的理由。

先从最显而易见的说起：钢笔——至少是带有分叉笔尖的钢笔——能实现其他笔类无法做到的事。由于钢笔能自然沉积不同墨量，并根据运笔速度和方向产生粗细变化的笔画，它为书写赋予了人性化的"特质"。相比圆珠笔、中性笔或凝胶笔，钢笔更能体现个人风格。在这个计算机主导的冷漠时代，这点弥足珍贵。

![Writer using fountain pen](https://www.richardspens.com/images/ref/ttp/whats_point/fountain.jpg) |  ![](https://www.richardspens.com/images/pixel.gif)  

如果你在1970年后就读于美国公立学校，很可能你是用圆珠笔学习写字的。你或许会对全美最大教师工会"国家教育协会"的研究结果感兴趣：该研究发现圆珠笔是导致书写潦草的重要因素。

没错，你没看错。圆珠笔会毁了你的字迹。原因何在？想象当你的手掌和前臂平放桌面时，手指通常自然微曲，拇指与食指伸展幅度最大。此时握笔姿势应如首图所示。

这种手指放松、不紧捏笔杆的姿势，正是书法专家所倡导的——我在1950年代学习帕尔默书写法时就被如此教导。正确姿势下，钢笔只是轻倚指间，笔尖几乎不需施压就能在纸面滑行。此时手指不应扭动，从象征意义来说，笔杆、手掌与前臂应如铁柱般稳固，动作源自肘部与肩部发力。我的老师常在练习时巡视教室，偶尔会从学生肩后突然抽笔——若笔杆毫无阻力地被抽离，说明学生握笔过紧。

这与多数人使用圆珠笔类产品的握姿（如次图所示）有何不同？首先，使用圆珠笔及其衍生品（包括几乎不费力的凝胶笔）时，你必须施加足够压力产生摩擦。没有这种关键摩擦力，滚珠无法转动，墨水难以流出。中性笔所需力度小于普通圆珠笔，凝胶笔更甚，但三者都需要一定程度的下压力。为此你不得不更紧握笔杆。我们都见过有人写字时食指末节内扣、指节发白的场景。这种姿势既疲劳又紧张，不仅损害健康，更影响书写——因为紧张状态会削弱精细动作控制能力，使你难以精准运笔。这些微妙影响日积月累，最终可能导致令人苦恼的书写痉挛。正确使用钢笔的人从不会受此困扰。

![Writer using fountain pen](https://www.richardspens.com/images/ref/ttp/whats_point/ballpoint.jpg)  

圆珠笔还有另一微妙特性：其滚珠被压嵌在笔芯末端。为防止滚珠脱落设计的金属环边，迫使你必须以较高角度持笔，否则就会感受到金属边刮擦纸面的不适感。紧握姿势与高角度持笔的双重作用，对书写质量堪称灾难。当然有人能克服这些写出漂亮字迹，但这本是不必要的额外努力。

最具说服力也最出人意料的理由来自医学界：越来越多医生开始建议关节炎、腕管综合征（CTS）或各类重复性劳损（RSI，从神经损伤到肌腱炎）患者使用钢笔。这些病症会使握压动作（如写字）产生不适甚至疼痛。而钢笔既不需紧握也不需下压，这让我那患有轻度手部关节炎的妻子从嘲讽者变成了钢笔使用者。

开篇时我列举了几个升级BiC圆珠笔的理由。事实证明，钢笔比圆珠笔更能满足这些需求：你可以用更优美的字迹赢得赞赏，从数百种墨水颜色中挑选最配服饰的款式，同时在书写时感受更舒适、更健康的状态。那么，你还有什么理由不用钢笔呢？
