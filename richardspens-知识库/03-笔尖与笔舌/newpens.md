# 半随机思考：现代钢笔是否如老款般优秀？  
_本文是首次发表于2008年4-5月刊《Stylus》杂志文章的更新扩展版_

---

在钢笔世界里，真正的新事物并不多。虽然我们能看到数十种新款式、新版本和新饰面，但这些美往往流于表面。所有新笔都基于相同的供墨技术——这项技术近半个世纪以来并无重大革新。少数公司仍在自主生产笔尖、笔舌（feed）等部件，但多数品牌采用Bock、JoWo和Schmidt等供应商的通用零件。收藏老钢笔的人认为Parker Premier不过是Parker 75的放大加重版，因其内部设计完全一致。（同理，现代Premiere实质是经过华丽包装的Waterman Philéas。）那么为何我们不将Delta Return to China限量款、Bexley Monarch或Conklin Mark Twain视为基础"Bock"或"JoWo"钢笔的变体？这是个有趣的悖论。

但本文目的并非探讨无解之谜。让我们尝试解析现代钢笔同质化的根源。

采用通用"内核"的首要原因是经济考量。多数钢笔公司规模较小，无力像昔日巨头那样投入资金组建大型研发部门，或建立专门生产线来生产相对少量的笔尖、笔舌等部件。"少量"是什么概念？Montblanc的作家系列限量版产量达2万支以上——这数字很可能超过Bexley自1993年创立以来的总产量。

![Fountain pen](https://www.richardspens.com/images/ref/ttp/newpens/dostoevsky.jpg)  
![kafka](https://www.richardspens.com/images/ref/ttp/newpens/kafka.jpg)

另一个原因在于现代人比半个世纪前的祖辈更挑剔。我们无法容忍钢笔轻微晃动就可能漏墨（过去人们只是学会避免晃动钢笔）。现代标准化笔舌的设计优先考虑防漏墨——即便牺牲书写可靠性和流畅度。结果如何？我们得到的是出水吝啬的笔、起笔困难的笔、飞白的笔。而我们容忍这些缺陷，只因更害怕重蹈那个杜撰的Waterman传说：Lewis Waterman因签约时钢笔漏墨而错失大单。

![Ink blot](https://www.richardspens.com/images/ref/ttp/newpens/splat.jpg)

更糟的是，如今许多笔尖笔舌制造者似乎并不真正懂行。这话或许尖锐，但随机检查百支全新现代钢笔的笔尖，你会发现几乎所有笔尖的铱粒都紧密贴合。在突破"两个物体不能同时占据同一空间"的物理定律前，这种设计注定导致墨水无法通过铱粒间隙接触纸面。有趣的是，百支随机老钢笔也可能出现同样状况——但这有合理原因：老钢笔使用硬橡胶笔舌，长期受压会导致变形。当笔尖持续下压，笔舌逐渐屈服，笔尖随之闭合。这些钢笔出厂时铱粒间本有间隙：文书用笔间隙较大，弹性笔尖间隙较小。现代制造商似乎忘记了这个基本设计准则。

硬币的另一面是：即便铱粒间距正确，许多现代笔尖[仍书写不佳](<ref/nibs/basics.htm> "阅读本文")。廉价钢笔往往刮纸，因其笔尖未精细打磨或切割铱粒后留有毛边；高端钢笔通常顺滑但易出现起笔困难——除非使用者像用圆珠笔般用力下压，这是由于过度抛光导致铱粒切割面过度圆润，形成"婴儿臀部"状凹陷。无论贵贱，新笔都可能存在铱粒错位问题。下图展示这些典型问题：

现代钢笔的问题不仅限于笔尖。笔舌不仅被设计成不惜代价防漏墨，选用廉价注塑塑料本身也带来难题。塑料是[疏水性](<ref/gloss/H.htm#hydrophobic>)材料，会排斥而非引导墨水。早期塑料笔舌在技术改良前确实可靠性欠佳。如今知名厂商通过等离子蚀刻使笔舌表面呈现类似硬橡胶的微孔结构，这类产品已与老式硬橡胶笔舌同样可靠。前文"老式"一词暗示现代硬橡胶笔舌也可能不尽人意——尽管硬橡胶本是近乎完美的笔舌材料。虽然历史经验为现代制造商指明了优秀笔舌的几何标准，但多数现代硬橡胶笔舌并未继承传统设计。这种偏离可能源于前文提到的漏墨恐惧：窄而浅的导气槽成为标配，单墨水通道被视为足够。更甚者，某些气流路径设计堪称怪异——例如当前Bock 8号笔尖搭配的笔舌，其异常浅的导气槽在笔尖中部就戛然而止，导致空气无法从气孔顺畅流通至导气槽。这类笔舌组合常在使用者连续书写时断墨。

但希望尚存。少数深谙笔尖原理并坚持调整所售钢笔的经销商（如同昔日一线品牌的做法），已在懂行的消费者群体中赢得了可观市场份额。图为[IndyPenDance](<http://www.indypendance.com/> "访问网站")的Linda Kennedy正在为客户调整笔尖。

![Nib expert tuning a nib](https://www.richardspens.com/images/ref/ttp/newpens/linda_tuning.jpg)

在高端领域，极少数公司愿为顶级笔尖投入重金，额外采购出厂即完美调校的笔尖。

当你仔细思考时，会发现这正是一支笔应有的本质。当然，总会有买家纯粹为了赏心悦目而收藏，但既然你在阅读本文，就说明你是个认真的用笔人——否则你根本不会读到这里。会哭的孩子有奶吃，如果有足够多的人持续发声，那些生产"中看不中用"钢笔的品牌方就不得不重视这个问题。
