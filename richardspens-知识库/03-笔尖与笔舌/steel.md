# 偷走钢材的魅力
> 来源：[Richard's Pens](https://www.richardspens.com/ref/ttp/steel.htm)
# 直击要点：偷走钢材的魅力  
（本文发布于2005年8月1日）  
本文是修订版，首次刊登于2004年12月/2005年1月出版的《Stylus》杂志。

---

Aurora Ipsilon、Columbus Academia、Filcao Nobile、Levenger True Writer和Waterman Philéas这些钢笔，除了都是性价比极高的优质钢笔外，还有什么共同点？不是价格，也不是颜色、尺寸、笔身材质或上墨系统。它们的共同之处在于都采用了钢制笔尖。

在这个消费者将黄金（越多越好）等同于品质的世界里，钢材似乎显得廉价且书写体验糟糕。还很难看。然而这些都不是事实，本文旨在破除这些误解。

**误解一：钢尖很廉价。**  
钢尖确实比金尖成本低，但"价格亲民"不等于"廉价劣质"。当然，质量低劣、没有硬质铱粒的钢尖确实存在。如今这类通常是IPG笔尖（"IPG"指这些欺骗性笔尖上刻的[IRIDIUM POINT GERMANY](<ref/gloss/I.htm#iridium_point_germany>)标记）。这类笔尖常见于某些中国产钢笔和手工车制钢笔的通用套件中，但相比70年前已大幅减少——因为钢笔已凭借自身书写价值立足，而非作为圆珠笔的竞品。与承载它们的钢笔一样，现代钢尖早已从技术进步中获益。

如果你花费超过20美元购买新钢笔，完全可以期待它配备质量可靠（即便不算卓越）、经久耐用的铱粒笔尖。稍高价位就能获得优质笔尖。仔细看看Waterman Philéas（市价约35美元）：你会看到双色钢尖造型优美、做工精良，通常书写顺滑且出墨流畅。随着价格上升，品质只会更好。

**误解二：钢尖书写糟糕。**  
多数钢尖偏硬，有些确实僵硬。但用过经典款Parker Duofold或Waterman Edson的人都知道，这些金尖钢笔硬如铁钉——却书写美妙。现代钢尖同样如此。事实上，钢尖开箱即用的表现往往优于我们推崇的那些华丽18K金尖。

这种反差源于钢尖通常全机械制造，而金尖——符合其奢侈品定位——常经手工打磨。手工打磨可能优于机械加工，但效率低且一致性差。机制钢尖更稳定，因此书写表现更均衡可靠。手工打磨还成本更高：要让笔尖滑行流畅（又不能太滑导致飞白）的工艺既罕见又昂贵。如果钢尖书写不佳，完全可以像金尖一样调整。即便请专业笔尖师调校钢尖钢笔，总花费通常仍低于同款金尖钢笔——后者同样可能需要调校。

**误解三：钢尖很难看。**  
丑陋并非某种金属的专属特质。钢尖确实可能难看，但也完全可以像金尖一样迷人。

以下是两支定制钢尖钢笔：Levenger True Writer和Columbus Academia：

![Nib](https://www.richardspens.com/images/ref/ttp/steel/truewriter.jpg)![Nib](https://www.richardspens.com/images/ref/ttp/steel/academia.jpg)

经过抛光（如图）后，TrueWriter的无镀层钢尖看起来完全像钯镀金尖，而Academia的镀金尖与18K金尖几乎无异。就像金尖一样，钢尖也能定制得赏心悦目。TrueWriter的笔尖被改磨成1.1毫米草书斜尖，适合签名时展现戏剧性笔锋；Academia则配备0.6毫米定制钝尖，日常书写略带风采。这些精心打磨的钢尖不仅外观媲美金尖，书写表现也毫不逊色。它们将以远低于金尖的价格，为主人提供终身愉悦的书写体验。

我们多数人都钟情金尖。这无可厚非——我也爱我的金尖。但那些完全排斥钢尖的人，错过了许多优质的书写体验。不妨在收藏中加入几支钢尖钢笔：它们是可靠的工作马，也能像金尖兄弟一样引人注目。
