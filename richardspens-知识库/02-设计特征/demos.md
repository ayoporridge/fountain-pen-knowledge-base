# 示范钢笔  
> 来源：[Richard's Pens](https://www.richardspens.com/ref/design/demos.htm)  
# 设计特点：示范笔  
### 问题所在  

现在是1925年。你是一家钢笔店的销售员，或是纽约梅西百货、费城沃纳梅克等知名百货公司钢笔专柜的店员。你正试图说服顾客，你销售的钢笔制作精良、性能可靠。但更重要的是，他们想知道这些钢笔的工作原理。这支笔如何上墨？那支笔如何保证墨水流畅？你当然不能拆开钢笔展示内部结构，因为那样就无法再销售了。你该怎么办？

如果你销售的是Bonded或Transo球囊上墨钢笔，或是Dunn-Pen的泵式上墨钢笔，事情就简单多了。尽管这些钢笔主要由不透明的赛璐珞制成，但它们的笔杆是透明的，你可以看到用于上墨的导气管。你甚至可以演示如何上墨；事实上，你可能专门留有一支笔用于演示。这里展示的第一支笔是Postal Reservoir钢笔，由生产Bonded和Transo的同一家公司制造，除品牌名外完全相同。第二支笔是现代复刻版的Dunn-Pen "Tattler"，其特色正是透明笔杆以展示墨水量。

其他少数制造商也采用了这种技术。Parker曾短暂推出过采用透明酚醛树脂笔杆的按钮上墨钢笔，Dunn-Pen也是如此；但透明酚醛树脂非常脆弱，钢笔跌落时极易破裂。Dunn-Pen后来改用赛璐珞材质，而Parker则暂停生产透明笔杆，直到1933年推出Vacumatic系列（本文稍后将讨论）。  

### 解决方案  

然而，对于驼峰式和杠杆式上墨钢笔来说，情况就没那么简单了。它们完全不透明，内部结构无从观察。早在20世纪初，这个问题通过切割式示范笔得到了解决。切割式示范笔是在普通钢笔的笔杆或笔帽（或两者）上切割出槽口或孔洞的产物。这些开口经过精心设计，以展示钢笔的独特功能。许多情况下，切割式示范笔是不完整的，缺少笔尖和笔舌（feed）。部分示范笔（尤其是插拔式笔帽款）甚至没有笔帽，因为普通插拔式笔帽内部并无特别结构可展示。下图是一支4号尺寸的Conklin Crescent-Filler锥形笔帽笔杆，可能是工厂次品，被改造成示范笔而非直接丢弃。
