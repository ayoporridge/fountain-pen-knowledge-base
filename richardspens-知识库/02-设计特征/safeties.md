# 安全钢笔  
> 来源：[Richard's Pens](https://www.richardspens.com/ref/design/safeties.htm)  
# 设计特点：安全钢笔  
### 问题根源  

**早期钢笔极易在使用者口袋中漏墨。** 第一代制造商采用硬橡胶制作笔身，并配备两种通用滑动笔帽：直筒笔帽（下图上方所示）——可套在笔杆两端的圆柱形区域上（用于保护笔尖或书写时反插笔帽），以及锥形笔帽（下图下方所示）——其内孔呈锥形而非圆柱形。许多锥帽钢笔的笔杆仅略小于笔帽开口以实现反插，但部分笔杆确实会逐渐收细以获得更好的反插效果，如图中Beaumel钢笔所示。

硬橡胶滑动笔帽并不完全可靠。随着使用，直筒笔帽会逐渐磨损变松，当钢笔置于衬衫口袋或手提包中时可能意外脱落。而锥形笔帽除非用力套在锥形笔杆上，否则根本不够紧密。但若使用者过度用力，笔帽可能开裂。若笔帽意外松脱，笔尖朝上携带仍有风险；若以其他方向（如笔尖朝下）携带则风险更大——当钢笔受到晃动时，储墨器可能向笔帽内漏墨。此外，多数早期钢笔的笔帽设有通气孔，即使笔帽紧闭也可能直接漏墨到使用者衣物上。  

### 解决方案  

解决滑动笔帽的可靠性问题是打造"安全"钢笔的第一步。19世纪最后十年开始出现的螺纹笔帽是一种解决方案。首款安全钢笔是1894年推出的Horton防漏墨水笔：
